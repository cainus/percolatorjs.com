Percolator is a new kind of web application framework for node.js
for fun and fast development of quality web APIs.

It's great for:

  • quality, public-facing APIs
  • single page apps
  • mobile apps

Great Status Responses

A great API should be concerned with more than just the happy path. Percolator does its best to help make error messages and other statuses simple for you to produce and simple for your clients to consume.

View details »

Just HTTP

Percolator tries to get out of HTTP's way by not adding unnecessary abstractions on top of it. If you know HTTP, you will not have to learn a bunch of new concepts to understand Percolator.

It doesn't concern itself with "views" or "models" or anything else that is peripheral to HTTP as an application protocol.
